# Assets Guide

## Where to Place Your Files

### Logo
Place your logo file at: `public/logo.png`
- Recommended size: 200x200px or similar square/rectangular format
- Format: PNG with transparent background works best
- The navbar will automatically display it

### Favicon
Place your favicon at: `public/favicon.ico`
- Recommended size: 32x32px or 16x16px
- Format: .ico file
- This appears in the browser tab

### Additional Assets
You can also use the `public/assets/` folder for any other images or files you need.

## Current Structure
```
public/
├── assets/          # Additional assets (optional)
├── images/          # Hero and content images
│   └── hero-image.jpg
├── logo.png         # Your logo (add this)
└── favicon.ico      # Your favicon (add this)
```
